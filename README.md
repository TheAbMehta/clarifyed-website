# Clarifyed.ai

**Whiteboard-native AI tutoring. No chatbot. Real learning.**

Clarifyed is the first AI tutoring platform where learning happens on a whiteboard, not in a chat window. The AI writes notes, draws diagrams, embeds interactive widgets, and teaches concepts stroke by stroke — then hands the pen to the student. When you make a mistake, it corrects you in real time.

## What Makes Clarifyed Different

Every AI tutor on the market is a chatbot. Student types a question, AI types back a paragraph. That's not how learning works.

Clarifyed has no chat interface. The AI teaches on a **multi-page digital whiteboard** — writing in handwritten style, drawing diagrams, embedding Desmos graphs, generating explainer videos, and saying "now you try." The student picks up the pen and solves on the same canvas.

### Core Capabilities

- **Whiteboard Canvas** — AI writes, draws, and teaches on a multi-page digital notebook. Both the AI and student work on the same surface.
- **Persistent Knowledge Graph** — Tracks everything a student has ever learned: topics, formulas, mastery levels, and misconceptions. The AI never forgets.
- **Real-Time Collaboration** — Students invite friends to the same whiteboard. The AI observes all participants and guides peer teaching.
- **Video Generation** — AI-generated 5-6 minute explainer videos with precise math animations, produced in ~3 minutes.
- **Interactive Widgets** — Embedded Desmos graphs, flashcards, MCQ quizzes, and mind maps — all inline on the canvas.
- **YouTube & PDF Integration** — Watch videos or open textbooks within the app and get AI help in context.

## Tech Stack

- **React 19** with TypeScript
- **Vite 6** for build tooling
- **Tailwind CSS v4** for styling
- **Framer Motion** for scroll-linked animations
- **Lucide React** for icons

## Development

```bash
npm run build     # Production build to dist/
npm run lint      # Type-check (tsc --noEmit)
```
